# Shell Template

## Intent

Use this for the real backend shell only.

## Structure

- Top header
- Side navigation
- Breadcrumb
- Main content
- User action area

## Notes

- Keep content width stable.
- Keep navigation and action zones visually quiet.
- Use token-driven surfaces and borders.

## Do

- Use `Layout`, `Menu`, `Breadcrumb`, `Card`, `Tabs`, `Tag`.

## Don’t

- Add marketing-style large hero blocks.
- Add decorative background art.
- Create a second shell style unrelated to TDesign.
